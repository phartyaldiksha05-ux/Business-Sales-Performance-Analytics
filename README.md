# 🛒 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an **interactive Power BI dashboard** built using Superstore sales data.  
The dashboard provides key business insights such as sales trends, profit analysis, category performance, and regional sales distribution to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales and profit performance  
- Identify top-performing and low-performing categories  
- Track sales trends over time (Month & Year)  
- Compare regional sales performance  
- Calculate profit margin and total orders  

---

## 📊 Key Metrics
- **Total Sales**
- **Total Orders**
- **Total Profit**
- **Profit Margin (%)**

---

## 📈 Visualizations Included
- Total Sales by Month & Year  
- Total Sales by Month, Year & Category  
- Total Profit by Category  
- Total Sales by Region  
- Top Products by Sales  
- Interactive slicers (Region, Order Date)

---

## 🛠 Tools & Technologies
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**
- **Data Visualization**

## 📌 Insights Summary
- East region contributes the highest sales  
- Technology category generates maximum profit  
- Furniture category shows lower / negative profit  
- Overall profit margin is approximately **15%**

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it using **Power BI Desktop**  
3. Interact with slicers to explore insights
